## 서비스

> "20대 청춘을 나라에 바치는 국군 장병들의 소중한 시간을  
> 더 **의미 있게** 만들 수 없을까?"

<br>

이 고민에서부터 프로젝트가 시작되었습니다.

<br>

짧다면 짧지만 길다면 긴,  
하루 동안의 **정비 시간**을 활용하여  
국군 장병들의 **자기개발**에 도움이 될 수 있는 프로그램을 만들었습니다.

<br>

- [**📽️ 시연 영상 보기**](https://www.youtube.com/watch?v=QQCqkgfOb_8)
