### Service
> "20대 청춘을 나라에 바치는 국군 장병들의 소중한 시간을 더 유의미하게 만들 수 없을까?"라는<br>
> 고민에서부터 프로젝트가 시작되었습니다.<br>
> 짧다면 짧지만 길다면 긴, <br>
> 하루동안의 개인 정비 시간을 활용하여 자기개발에 도움이 될 수 있는 프로그램을 만들었습니다.

<br>
